Presentación del programa

Al finalizar el programa Técnico en Desarrollo de Software, el egresado tendrá las competencias para asistir el desarrollo de aplicativos tipo web y para dispositivos móviles, utilizando herramientas, metodologías y lenguajes de programación requeridos por el entorno productivo, demostrando respeto, orden y creatividad en su desempeño.

Duración: 3 semestres

Modalidad de estudio: Presencial.

Jornada: Lunes a viernes, jornada diurna o nocturna.

Requisitos: Fotocopia del documento de identidad. Fotocopia del título de bachiller o del acta de grado. Estudiantes de 10º y 11º, presentar certificado de estudio.

Pensum

  • Semestre I
    • Lógica de programación.
    • Metodologías ágiles para la programación.
    • Introducción a la programación.
    • Cátedra ser emprendedor.
  • Semestre II
    • Promoción de aplicaciones móviles.
    • Programación para la web I.
    • Gestión de base de datos.
  • Semestre III
    • Nuevas tecnologías de programación.
    • Programación de aplicaciones móviles II.
    • Programación para la web II.
